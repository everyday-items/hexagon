## Hexagon 差异化优势

1. **Go 原生**: 单二进制部署、高并发、低资源占用
2. **六边形均衡**: 唯一在易用性、性能、扩展性、编排、可观测、安全六维均衡的框架
3. **渐进式复杂度**: 3 行入门 → 声明式配置 → 图编排
4. **生产优先**: 内置完整的可观测性和安全防护
5. **中文优先**: 文档和社区对国内开发者友好

## 技术对标

| 维度 | Hexagon 实现 | 对标方案 |

| 可观测 | 钩子 + OTel + Prometheus | LangChain Callbacks |
| 安全 | Guard Chain + RBAC + 成本控制 | Semantic Kernel Filters |
